Modularer Elektrobaukasten

Project Summary

This project is a modular electrical kit for vehicle related functions. The goal was to build a compact and flexible system that can connect different inputs and outputs in one structure.

The setup combines Raspberry Pi, Arduino boards, switches, sensors, and output elements such as lights, horn, heating, ventilation, and wipers. It is designed as a practical student project in electromobility and embedded systems.

  • Modular structure for easier extension and testing
  • Communication between controller boards and vehicle functions
  • Combination of software logic, electronics, and web display
