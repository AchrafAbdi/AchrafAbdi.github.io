ESP32 Motor Control with NPN Transistor

A small DIY electronics experiment where I tested if a DC motor can be controlled directly with an ESP32 and a simple NPN transistor, instead of using a relay.

← Back to Portfolio

Project Idea

The idea for this project came from a tutorial where a similar circuit was used to control a relay. In that tutorial, the NPN transistor switched the relay, and the relay controlled the motor.

I wanted to test whether I could use the same switching principle to control the motor directly through the transistor, without using the relay in between.

For this experiment I used an ESP32, an NPN transistor 8050, a base resistor, a DC motor and a flyback diode to protect the circuit from voltage spikes caused by the motor.

ESP32 NPN Transistor PWM DC Motor MicroPython

How the Project Works

  1. The ESP32 sends a PWM signal from GPIO 18.
  2. The PWM signal goes through a 1 kΩ resistor to the base of the NPN transistor.
  3. The transistor works like an electronic switch and controls the current flow through the motor.
  4. The flyback diode protects the circuit when the motor is switched off.
  5. With MicroPython, the motor runs for a few seconds and then stops again.

What I Learned

This project helped me understand how a transistor can be used as a switch in a real circuit. I also learned why protection components like a flyback diode are important when working with motors and other inductive loads.

It was a simple experiment, but it connected several important topics together: ESP32 programming, PWM control, transistor switching, motor control and basic circuit protection.
