ESP32 Motor Control with PWM

Full project view with circuit, code and project explanation.

Circuit Design

The circuit uses an ESP32, an NPN transistor 8050, a resistor, a DC motor and a flyback diode.

MicroPython Code

The ESP32 sends a PWM signal to control the motor speed.

Project Description

In this DIY project I wanted to test if I could control a DC motor directly with a simple NPN transistor instead of using a relay module.

I got the idea from a tutorial where a similar circuit was used. In that tutorial the NPN transistor controlled a relay and the relay controlled the motor.

I thought that the same switching principle could maybe also be used to control the motor directly through the transistor itself.

For the experiment I used:

  • ESP32
  • NPN transistor 8050
  • DC motor
  • Flyback diode
  • PWM signal

Summary

This project helped me understand how a transistor can be used as an electronic switch in a real circuit.

I also learned how PWM can be used to control motor speed and why a flyback diode is important when working with motors and inductive loads.

ESP32 I²C Communication

PDF documentation of the I²C communication between three ESP32 boards.

Project Description

In this DIY project, three ESP32 boards were connected using the I²C bus. Two ESP32 boards were configured as slaves with the addresses 0x08 and 0x09. These two slave boards were programmed in C++ using the Arduino IDE.

The third ESP32 was used as the master and was programmed in Python using Thonny. The master used the MicroPython machine.I2C library with GPIO21 as SDA and GPIO22 as SCL.

After scanning the I²C bus, the master successfully detected both slave devices. The output showed the addresses 8 and 9, which confirmed that the I²C connection between the three ESP32 boards was working correctly.

Summary

This project helped me understand the basic structure of I²C communication with one master and multiple slaves. It also helped me practice using different programming environments in one small embedded system project.

  • ESP32 master programmed with Python in Thonny
  • ESP32 slaves programmed with C++ in Arduino IDE
  • I²C bus communication
  • Slave addresses 0x08 and 0x09
  • GPIO21 used as SDA
  • GPIO22 used as SCL
