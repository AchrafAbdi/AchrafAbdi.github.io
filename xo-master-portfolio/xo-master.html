XO Master - NAO Tic Tac Toe Project

An interactive Tic Tac Toe project with the humanoid NAO robot, using visual marker recognition, Choregraphe behavior control, Python game logic and physical human robot interaction.

Back to Portfolio

Project Summary

XO Master is an interactive Tic Tac Toe system where the NAO robot analyzes a physical game board with visual markers, stores the board state in an internal 3x3 matrix and communicates its next move to the human player through speech output.

The project combines NAO camera usage, NAOMarker detection, Choregraphe behavior boxes and Python scripts. Instead of voice input, the interaction is one directional: NAO observes the board, evaluates the marker IDs and tells the user what to do next.

  • Physical Tic Tac Toe board with NAO markers
  • Marker IDs for empty fields, X fields and O fields
  • Python based game logic with internal 3x3 matrix
  • Choregraphe behavior structure for scanning and marker processing
  • Robot positioning and distance calibration for reliable recognition
  • Speech output for human robot interaction
