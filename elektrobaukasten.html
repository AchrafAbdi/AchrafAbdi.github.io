Modularer Elektrobaukasten

A Multi-Controller Embedded System for Vehicle Functions

Project Overview

This project serves as a modular electrical testing kit for electromobility applications. It simulates essential vehicle functions like lighting, ventilation, and sensor monitoring within a unified control structure.


Key Technical Highlights:

  • Master-Slave architecture using I2C communication.
  • Raspberry Pi acting as the main processor for high-level logic.
  • Dual Arduino Mega boards handling real-time I/O (switches, sensors, actuators).
  • Integration of heating, ventilation, and lighting controls.
