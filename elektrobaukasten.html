Project Summary

This project was developed as a modular electrical kit for vehicle related functions. The goal was to create a compact and flexible system that can connect different inputs and outputs in one practical structure.

The setup combines Raspberry Pi, Arduino Mega boards, switches, sensors, and output elements such as lighting, horn, heating, ventilation, and wipers. The project connects software logic, electronics, and communication between multiple components.

  • Modular design for easier extension and testing
  • Communication between Raspberry Pi and Arduino boards
  • Integration of sensors, switches, and output functions
  • Practical application in electromobility and embedded systems
